# MaxiTerm Relay (à venir)

Le **relais MaxiTerm** est un serveur **open source et auto-hébergeable**
(prévu en [Vapor](https://vapor.codes)) permettant des fonctions avancées
(ex. accès derrière NAT, push, partage de session) **sans aucun verrou côté
client**.

## Principe directeur

- Le relais est **gratuit et auto-hébergeable** (Mac mini, VPS, etc.).
- Un éventuel **service hébergé géré** pourra être proposé en abonnement : c'est
  de l'**hébergement/commodité**, jamais une fonctionnalité verrouillée de l'app.
- Tout reste **faisable gratuitement en self-host**.
